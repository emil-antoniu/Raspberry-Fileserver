# ARC-FILESERVER

<p>Baseline configurations for the fileserver of Andromeda Research Center Inc.</p>

<p>The fileserver is a Raspberry Pi Model 3 A+ provisioned with a 0.96 Inch I2C/IIC 4-Pin OLED Display Module. Cabled LAN is established via its USB 2.0 port.</p>

<p>Configurations are made with the default user in mind. Connections are done via SSH with public-key authentication only.</p>

<p>Carefully read the documented steps to ensure that the software and configurations are relevant and up-to-date.</p>
